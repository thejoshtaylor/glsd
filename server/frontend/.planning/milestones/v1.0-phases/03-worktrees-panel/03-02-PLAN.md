Component structure — `export function Gsd2WorktreesTab({ projectId }: Gsd2WorktreesTabProps)`:

**State:**
- `const [expandedRows, setExpandedRows] = useState<Set<string>>(new Set())` — tracks which rows are expanded (multiple may be open)
- `const [removeTarget, setRemoveTarget] = useState<WorktreeInfo | null>(null)` — which worktree the AlertDialog is confirming removal for

**Hooks:**
- `const { data: worktrees, isLoading, isError } = useGsd2Worktrees(projectId)`
- `const removeMutation = useGsd2RemoveWorktree()`

**Toggle function:**
```typescript
const toggleRow = (name: string) => {
  setExpandedRows(prev => {
    const next = new Set(prev);
    if (next.has(name)) next.delete(name);
    else next.add(name);
    return next;
  });
};
```

**Confirm remove function:**
```typescript
const confirmRemove = () => {
  if (!removeTarget) return;
  removeMutation.mutate({ projectId, worktreeName: removeTarget.name });
  setRemoveTarget(null);
};
```

**Render structure (follow UI-SPEC exactly):**

Loading state — Card with CardHeader (`pb-3`) containing `<GitBranch className="h-4 w-4" />` + "Worktrees" title (text-sm font-semibold), CardContent with 3x `<Skeleton className="h-12 w-full" />` in `space-y-3`.

Error state — Same Card/CardHeader, CardContent with `py-4 text-center text-sm text-status-error` and copy: "Failed to load worktrees — check that the project path is accessible."

Empty state (worktrees is empty array) — Same Card/CardHeader, CardContent with `py-12 text-center`:
- Heading: `<p className="text-sm font-medium text-muted-foreground">No active worktrees</p>`
- Body: `<p className="text-xs text-muted-foreground mt-1">GSD-2 creates these automatically when running parallel slices.</p>`

Data state — Card with header, then `<div className="divide-y">` containing one row per worktree:

Each row is a `<div>` containing:
- **Row header** (`py-3 px-4 flex items-center gap-2 cursor-pointer` — click on this area toggles accordion, but NOT on the Remove button):
  - Chevron toggle: `<Button variant="ghost" size="icon" className="h-6 w-6 shrink-0" aria-label={\`${expandedRows.has(wt.name) ? 'Collapse' : 'Expand'} ${wt.name} diff\`}>` containing `<ChevronRight className="h-4 w-4 transition-transform duration-200" />` with `style={{ transform: expandedRows.has(wt.name) ? 'rotate(90deg)' : 'rotate(0deg)' }}`
  - `<GitBranch className="h-4 w-4 shrink-0 text-muted-foreground" />`
  - Name: `<span className="text-sm font-mono font-medium text-foreground">{wt.name}</span>`
  - Branch badge: `<Badge variant="outline" className="font-mono text-xs">{wt.branch}</Badge>`
  - Change counts (flex-1, text-right or use `ml-auto`): `<span className="text-xs">` containing:
    - `<span className="text-status-success">+{wt.added_count} added</span>`
    - ` · ` in `text-muted-foreground`
    - `<span className="text-status-warning">{wt.modified_count} modified</span>`
    - ` · ` in `text-muted-foreground`
    - `<span className="text-status-error">{wt.removed_count} removed</span>`
  - Remove button (stop propagation on click to prevent row toggle): `<Button variant="destructive" size="sm" onClick={(e) => { e.stopPropagation(); setRemoveTarget(wt); }} disabled={removeMutation.isPending}>` with `<Trash2 className="h-3.5 w-3.5" />` (or `<Loader2 className="h-3.5 w-3.5 animate-spin" />` when mutation is pending for this specific worktree)

- **Path row** below the main row items: `<span className="text-xs font-mono text-muted-foreground truncate" title={wt.path}>{wt.path}</span>`

- **Expanded diff section** (only rendered when `expandedRows.has(wt.name)` is true):
  - Container: `<div className="border-t px-4 py-3 bg-muted/30">`
  - Render a `<WorktreeDiffSection projectId={projectId} worktreeName={wt.name} />` sub-component (can be inline or extracted)
  - This sub-component uses `useGsd2WorktreeDiff(projectId, wt.name, true)` — the `enabled` flag is always `true` when rendered because parent only renders it when expanded
  - Loading: `<Skeleton className="h-8 w-full" />`
  - Error: `<p className="text-xs text-status-error">Failed to load diff</p>`
  - Data: Three groups (only render groups with count > 0):
    - **Added** group: `<div className="mb-2">` with header `<div className="flex items-center gap-1 text-status-success text-xs font-medium mb-1"><FilePlus className="h-3 w-3" /> Added ({diff.added_count})</div>`, then file list: `diff.added.map(f => <div className="text-xs font-mono text-status-success/80 py-1 pl-6" key={f}>{f}</div>)`
    - **Modified** group: same pattern with `text-status-warning`, `FilePenLine` icon
    - **Removed** group: same pattern with `text-status-error`, `FileMinus` icon
  - Wrap file lists in `<ScrollArea className="max-h-48">` when count > 8

**AlertDialog** (rendered once at the bottom of the component, controlled by `removeTarget` state):
```jsx
<AlertDialog open={!!removeTarget} onOpenChange={(open) => !open && setRemoveTarget(null)}>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Remove {removeTarget?.name}?</AlertDialogTitle>
      <AlertDialogDescription>
        This will delete the worktree and branch {removeTarget?.branch}.
      </AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel>Cancel</AlertDialogCancel>
      <AlertDialogAction onClick={confirmRemove} className="bg-destructive text-destructive-foreground hover:bg-destructive/90">
        Remove
      </AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```
