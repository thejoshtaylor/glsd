```rust
// Emit GSD-2 events for .gsd/ file changes
if changed_path.contains("/.gsd/")
    || changed_path.contains("\\.gsd\\")
{
    // Skip .gsd/worktrees/ to prevent event storm during builds in worktrees
    if !changed_path.contains("/.gsd/worktrees/")
        && !changed_path.contains("\\.gsd\\worktrees\\")
    {
        let change_type =
            if changed_path.contains("/STATE.md")
                || changed_path.contains("\\STATE.md")
            {
                "gsd2_state"
            } else if changed_path.contains("/milestones/")
                || changed_path.contains("\\milestones\\")
            {
                "gsd2_milestone"
            } else if changed_path.contains("metrics.json") {
                "gsd2_metrics"
            } else {
                "gsd2_other"
            };
        let _ = app_handle.emit(
            "gsd2:file-changed",
            serde_json::json!({
                "project_path": project_path_clone,
                "file_path": changed_path,
                "change_type": change_type,
            }),
        );
    }
}
```

**CRITICAL:** The `.gsd/worktrees/` exclusion is essential. Worktrees contain full project copies with node_modules, target/, etc. Without this filter, `pnpm install` or `cargo build` in a worktree would generate millions of events per second.

**CRITICAL:** Place the GSD-2 block BEFORE the `.planning/` block so that `.gsd/` paths are matched first. A path cannot match both (they are different directories), but ordering makes intent clear.

**CRITICAL:** Use the SAME `project_path_clone` and `app_handle` variables already captured in the closure. Do NOT create new clones.
