GSD-2 ROADMAP.md Slice Format (from RESEARCH.md Pattern 6):
```markdown
## Slices
- [ ] **S01: Title Here** `risk:low` `depends:[S00]`
  > After this: description
- [x] **S02: Done Slice** `risk:medium`
```
Regex: `^\s*-\s+\[([ xX])\]\s+\*\*([\w.]+):\s+(.+?)\*\*\s*(.*)`

GSD-2 PLAN.md Task Format (from RESEARCH.md Pattern 7):
```markdown
## Tasks
- [ ] **T01: Task Title** `est:2h`
  description line
  - Files: `src/foo.rs`, `src/bar.rs`
  - Verify: run `cargo test`
- [x] **T02: Done Task** `est:30m`
```
Regex: `^\s*-\s+\[([ xX])\]\s+\*\*([\w.]+):\s+(.+?)\*\*\s*(.*)`

Add two new command functions to `src-tauri/src/commands/gsd2.rs`:

**1. `gsd2_derive_state`:**
```rust
#[tauri::command]
pub async fn gsd2_derive_state(
    db: tauri::State<'_, DbState>,
    project_id: String,
) -> Result<Gsd2State, String> {
```

Algorithm (simplified from state.js `deriveState()`):
1. Get project path from DB
2. Read `.gsd/milestones/` directory, sort by ID
3. For each milestone:
   a. Resolve ROADMAP.md via three-tier file resolver
   b. Parse slices from `## Slices` section
   c. For each slice: resolve PLAN.md, parse tasks from `## Tasks` section
   d. Count done/total for slices and tasks
   e. Mark milestone as complete if ALL slices are done
4. Find the FIRST non-complete milestone = `active_milestone_id`
5. Within the active milestone, find the FIRST non-done slice = `active_slice_id`
6. Within the active slice, find the FIRST non-done task = `active_task_id`
7. Read `.gsd/STATE.md` frontmatter for `phase` value (if exists)
8. Sum all counts across milestones for totals
9. Return Gsd2State struct

**Important correctness rules:**
- Milestones MUST be sorted by ID (M001 before M002) before walking
- A milestone is "complete" when ALL its slices have `done == true`
- If all milestones are complete, `active_milestone_id` = None
- `active_slice_id` and `active_task_id` are only set when there IS an active milestone
- Count tasks across ALL slices of ALL milestones for totals (not just active milestone)

**Reading STATE.md:**
```rust
let state_path = Path::new(&project_path).join(".gsd").join("STATE.md");
let phase = if state_path.exists() {
    let content = fs::read_to_string(&state_path).unwrap_or_default();
    let (fm, _) = parse_frontmatter(&content);
    fm.get("phase").cloned()
} else {
    None
};
```
