Perform a systematic dead code audit of the frontend codebase. The audit methodology:

**Step 1: Audit exported React components in src/components/**
For every `.tsx` file in `src/components/` (recursively), check if the default or named exports are imported by any other file in `src/`. Use grep to search for the component name across the codebase. If a component is exported but never imported anywhere else, it is dead code — delete the file.

**Step 2: Audit hooks in src/hooks/**
For every file in `src/hooks/`, check if the exported hook function is imported anywhere in `src/`. If not, delete it.

**Step 3: Audit exported types/interfaces in src/lib/tauri.ts**
For every `export interface` and `export type` in `src/lib/tauri.ts`, check if the type name is referenced anywhere else in `src/` (outside tauri.ts itself). Types only used internally within tauri.ts are fine. Types that are exported but never imported elsewhere are candidates for removal — but be conservative: only remove types that are clearly orphaned (e.g., `Gsd2RoadmapProgressData` was already removed in plan 01).

**Step 4: Audit query hooks in src/lib/queries.ts**
For every exported `use*` hook in `src/lib/queries.ts`, check if it is imported by any component or page. If a hook is exported but never used, remove it.

**Step 5: Audit query keys in src/lib/query-keys.ts**
For every key in the `queryKeys` object, check if it is referenced in `src/lib/queries.ts` or any other file. Remove orphaned keys (keys not used by any hook).

**Step 6: Verify clean build**
Run `pnpm build` to confirm zero TypeScript errors after all removals. Run `pnpm test` to confirm zero test failures.

**Disposal policy (from CONTEXT.md):** Delete dead code entirely. No commented-out code, no archive folder. Git history preserves it.

**Important:** Do NOT remove code that is used only in tests (test utilities, test helpers). Do NOT remove types that are part of the public API surface of tauri.ts even if not currently used by a component (they may be used by the Rust backend contract).
