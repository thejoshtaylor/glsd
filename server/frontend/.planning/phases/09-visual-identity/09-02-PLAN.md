**Step 1: Create the SVG icon**

Write a new `src-tauri/icons/icon.svg` with these specifications:
- **Canvas**: 512x512 viewBox, pure black `#000000` rectangle filling the entire canvas
- **Text**: Bold "GSD" in white `#FFFFFF`, centered horizontally and vertically
  - Use a geometric sans-serif style — since SVG text rendering varies, use `<text>` with `font-family="Arial, Helvetica, sans-serif"` and `font-weight="800"` (extra-bold)
  - Font size approximately 180-200px so "GSD" occupies ~70% of canvas width
  - Vertically centered with slight upward bias to leave room for the accent underline
  - Letter spacing slightly expanded for readability at small sizes
- **Cyan accent**: A single horizontal line (underline) beneath the "GSD" text
  - Color: `#0cb4ce` (approximately hsl(189 94% 43%))
  - Width: approximately 60-70% of canvas width, centered
  - Height: approximately 6-8px
  - Positioned approximately 20-30px below the text baseline
- **No SVG-level border-radius** — Tauri handles macOS rounded corners automatically
- **No external font dependencies** — use system fonts only

**Step 2: Preview at 32x32**

Before generating all sizes, render a 32x32 preview to confirm legibility:
```bash
/opt/homebrew/bin/rsvg-convert -w 32 -h 32 src-tauri/icons/icon.svg -o /tmp/gsd-icon-32-preview.png
```
Visually confirm the "GSD" text is readable (check file exists and has non-zero size). If not legible, increase font weight or simplify the design.

**Step 3: Generate all PNG sizes**
```bash
/opt/homebrew/bin/rsvg-convert -w 512 -h 512 src-tauri/icons/icon.svg -o src-tauri/icons/icon.png
/opt/homebrew/bin/rsvg-convert -w 128 -h 128 src-tauri/icons/icon.svg -o src-tauri/icons/128x128.png
/opt/homebrew/bin/rsvg-convert -w 256 -h 256 src-tauri/icons/icon.svg -o src-tauri/icons/128x128@2x.png
/opt/homebrew/bin/rsvg-convert -w 32  -h 32  src-tauri/icons/icon.svg -o src-tauri/icons/32x32.png
```

**Step 4: Generate ICO (Windows)**
```bash
/opt/homebrew/bin/convert src-tauri/icons/icon.png \
  \( -clone 0 -resize 256x256 \) \
  \( -clone 0 -resize 128x128 \) \
  \( -clone 0 -resize 64x64 \) \
  \( -clone 0 -resize 48x48 \) \
  \( -clone 0 -resize 32x32 \) \
  \( -clone 0 -resize 16x16 \) \
  -delete 0 src-tauri/icons/icon.ico
```

**Step 5: Generate ICNS (macOS)**
```bash
mkdir -p /tmp/gsd.iconset
/opt/homebrew/bin/rsvg-convert -w 16   -h 16   src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_16x16.png
/opt/homebrew/bin/rsvg-convert -w 32   -h 32   src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_16x16@2x.png
/opt/homebrew/bin/rsvg-convert -w 32   -h 32   src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_32x32.png
/opt/homebrew/bin/rsvg-convert -w 64   -h 64   src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_32x32@2x.png
/opt/homebrew/bin/rsvg-convert -w 128  -h 128  src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_128x128.png
/opt/homebrew/bin/rsvg-convert -w 256  -h 256  src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_128x128@2x.png
/opt/homebrew/bin/rsvg-convert -w 256  -h 256  src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_256x256.png
/opt/homebrew/bin/rsvg-convert -w 512  -h 512  src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_256x256@2x.png
/opt/homebrew/bin/rsvg-convert -w 512  -h 512  src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_512x512.png
/opt/homebrew/bin/rsvg-convert -w 1024 -h 1024 src-tauri/icons/icon.svg -o /tmp/gsd.iconset/icon_512x512@2x.png
iconutil -c icns /tmp/gsd.iconset -o src-tauri/icons/icon.icns
rm -rf /tmp/gsd.iconset
```
