## Typography

| Role | Size | Weight | Line Height | Differentiation |
|------|------|--------|-------------|-----------------|
| Body | 14px | 400 | 1.5 | Default prose, paragraph text |
| Label | 14px | 600 | 1.4 | Same size as Body; distinguished by weight (600) and `--muted-foreground` color |
| Heading | 20px | 600 | 1.2 | Section and panel headings |
| Display | 28px | 400 | 1.1 | Large numeric or hero callouts |

Font weights in use: **400 (regular)** and **600 (semibold)** only. Maps to Tailwind `font-normal` and `font-semibold`. Weights 500 (medium) and 700 (bold) are not part of this contract and must not be introduced in this phase.

Font: `Inter var` / `Inter` as primary sans-serif. `JetBrains Mono Variable` for terminal and code surfaces. Font size presets (`.font-scale-sm` = 13px, `.font-scale-md` = 14px, `.font-scale-lg` = 16px) are retained from globals.css — no changes in this phase.

Label and Body share 14px size. Labels are distinguished from body text by weight (600 vs 400) and color (`--muted-foreground` for labels vs `--foreground` for body).

Source: tailwind.config.js fontFamily declarations + globals.css font-scale presets + checker revision 2026-03-21 (collapsed to 2 weights; Label size aligned to Body).

### 60/30/10 Color Split

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(0 0% 0%)` — pure black | `--background`: page background, main content area |
| Secondary (30%) | `hsl(0 0% 5%)` — near-black | `--card`, `--popover`: cards, sidebar, nav, elevated panels |
| Accent (10%) | `hsl(189 94% 43%)` — cyan | Buttons (primary), focus rings, active nav state, links, interactive element highlights, progress indicators |
| Destructive | `hsl(0 62.8% 30.6%)` | Destructive action buttons only |

Accent reserved for: primary buttons, focus rings (`--ring`), active/selected nav items, links, interactive element hover borders, progress bar fill, headless session running indicator. NOT used for general card borders or surface coloring.

## Icon Contract

### Delivery Spec

| File | Size | Format |
|------|------|--------|
| `src-tauri/icons/icon.svg` | Vector | SVG source — master file |
| `src-tauri/icons/icon.png` | 512×512 | PNG, generated from SVG |
| `src-tauri/icons/128x128.png` | 128×128 | PNG, generated from SVG |
| `src-tauri/icons/128x128@2x.png` | 256×256 | PNG, generated from SVG |
| `src-tauri/icons/32x32.png` | 32×32 | PNG, generated from SVG — must be readable at this size |
| `src-tauri/icons/icon.icns` | Multi-size | macOS bundle icon |
| `src-tauri/icons/icon.ico` | Multi-size | Windows icon |

### Icon Design Contract

- **Canvas**: Pure black square background `#000000`, no rounding at SVG level (Tauri handles macOS rounded corners)
- **Content**: Bold "GSD" letterform in white `#FFFFFF`, centered horizontally and vertically
- **Accent treatment**: A single cyan `hsl(189 94% 43%)` element — recommended: a thin cyan underline beneath the "GSD" text, or the letters themselves in cyan. One treatment only, not both.
- **Typeface feel**: Geometric sans-serif, weight equivalent to 700-800. Exact font at Claude's discretion.
- **Readability requirement**: At 32×32px, "GSD" text must be legible — no decorative elements that collapse at small size
- **Spacing**: "GSD" text occupies approximately 70% of canvas width, vertically centered with slight upward bias if accent underline is used
