    OS detection (per D-06):
    ```typescript
    function detectOS(): 'macos' | 'linux' | 'windows' {
      const ua = navigator.userAgent.toLowerCase();
      if (ua.includes('mac')) return 'macos';
      if (ua.includes('win')) return 'windows';
      return 'linux';
    }
    ```

    Install commands per OS tab (per D-07, D-08):
    ```
    macOS/Linux:
      curl -fsSL ${window.location.origin}/install | sh
      gsd-cloud login <CODE>
      gsd-cloud start

    Windows/WSL:
      # Run in WSL2 terminal
      curl -fsSL ${window.location.origin}/install | sh
      gsd-cloud login <CODE>
      gsd-cloud start
    ```

    Each command line gets a copy button using the existing `useCopyToClipboard` hook.

    Polling logic (per D-09):
    - On code generation, snapshot `initialNodeCount` from current `useNodes` data
    - Enable `refetchInterval: 3000` on useNodes query while `isPolling=true`
    - On each refetch, compare `data.count > initialNodeCount`
    - If new node detected: set `isPolling=false`, call `onOpenChange(false)`, show toast "Node '<name>' connected." (per D-10)
    - Use a `useEffect` with a 10-minute timeout. On expiry: set `isExpired=true`, `isPolling=false`. Show "Expired -- regenerate" state with a button that resets to step 1 (per D-10).

    For the polling integration, use a SEPARATE useQuery or modify useNodes to accept refetchInterval:
    ```typescript
    const { data: pollingData } = useQuery({
      queryKey: ['nodes'],
      queryFn: nodesApi.listNodes,
      refetchInterval: isPolling ? 3000 : false,
      enabled: isPolling,
    });
    ```
    Note: This shares the queryKey with useNodes, so React Query deduplicates. When isPolling is false and enabled is false, it stops polling but useNodes on the parent page still works normally.

    Actually, since both use queryKey ['nodes'], set refetchInterval conditionally:
    Better approach -- use the same useNodes() but pass polling state. Since useNodes is a simple wrapper, just use useQuery directly in the modal with the same key:
    ```typescript
    const nodesQuery = useQuery({
      queryKey: ['nodes'],
      queryFn: nodesApi.listNodes,
      refetchInterval: isPolling ? 3000 : false,
    });
    ```

    On modal close (via X or backdrop), set `isPolling=false` to stop polling (Pitfall 6).

    Reset all state when modal opens (in onOpenChange handler or useEffect on open prop).
