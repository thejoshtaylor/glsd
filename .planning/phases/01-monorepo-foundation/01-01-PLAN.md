
**Step 4: Move daemon and protocol-go into node/**
```bash
git mv daemon node/daemon
git mv protocol-go node/protocol-go
```

**Step 5: Move pnpm-lock.yaml to root**
The gsd-vibe lockfile needs to be at the workspace root for pnpm workspaces:
```bash
git mv server/frontend/pnpm-lock.yaml pnpm-lock.yaml
```

**Step 6: Update server/frontend/package.json name field**
Change the `"name"` field from `"vcca"` to `"server-frontend"` so pnpm `--filter` commands work with readable names. Do NOT change any other fields in this task.
