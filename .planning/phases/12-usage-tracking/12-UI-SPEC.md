## Layout Specification

### /usage Page Layout (top to bottom)

1. **PageHeader** — "Usage" with `BarChart3` icon, period `Select` as action slot
2. **Summary Cards Row** — 3 cards in a responsive grid (`grid-cols-1 sm:grid-cols-3 gap-4`)
   - Total Cost (`formatCost`)
   - Total Sessions (integer)
   - Avg Cost/Session (`formatCost`)
3. **Node Breakdown Card** — `Card` with `CardHeader` "Cost by Node", containing per-node rows:
   - Each row: node name (left), cost + session count (right), `Progress` bar below
   - Sorted by cost descending
4. **Daily Cost Chart Card** — `Card` with `CardHeader` "Daily Cost", containing recharts `BarChart`
   - Height: 240px (mobile), 300px (desktop)
   - X-axis: date labels (MMM DD format)
   - Y-axis: cost in USD
   - Bar fill: `hsl(var(--primary))`
   - Tooltip: date + `formatCost` value
5. **Session Table Card** — `Card` with `CardHeader` "Recent Sessions" + page indicator, containing:
   - Table columns: Date, Node, In Tokens, Out Tokens, Cost, Duration
   - 25 rows per page
   - Prev/Next buttons in `CardFooter`

### Mobile Responsiveness

- Summary cards stack vertically on `< sm` breakpoint
- Chart height reduces to 240px on mobile
- Session table scrolls horizontally on narrow viewports (`overflow-x-auto`)
- Node breakdown rows are full-width at all breakpoints

## Activity Feed Integration

Per D-08, `taskComplete` feed items display inline cost breakdown:

```
Task completed · in:900 out:312 · $0.04 · 42s
```

Format: `Task completed · in:{inputTokens} out:{outputTokens} · {formatCost(costUsd)} · {formatDuration(durationMs)}`

- Token counts: raw integers, no abbreviation (activity feed has space)
- Cost: uses `formatCost` utility
- Duration: uses `formatDuration` utility
- Entire breakdown in `text-xs text-muted-foreground font-mono` after the event label

## Interaction States

| State | Behavior |
|-------|----------|
| Page load | Summary cards and chart show `Skeleton` components; session table shows skeleton rows |
| Period change | All data re-fetches; skeleton states shown during load |
| Pagination click | Session table shows skeleton rows; scrolls to top of table card |
| Empty period | Summary cards show "$0.00" / "0" / "$0.00"; chart shows empty state text; table shows empty state |
| API error | Error message replaces content area (not a toast); retry via period re-select or page refresh |
| Hover on chart bar | Tooltip appears with date and formatted cost |
| Activity feed update | New `taskComplete` events append to feed with cost breakdown in real-time (via existing SSE) |
