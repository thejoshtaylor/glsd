**queries.ts changes (per D-01):**

1. Add import at top: `import * as projectsApi from './api/projects';` and `import type { ProjectPublic } from './api/projects';`
2. Replace `useProjectsWithStats` to call the real API:
```typescript
export const useProjectsWithStats = () =>
  useQuery({
    queryKey: queryKeys.projectsWithStats(),
    queryFn: async () => {
      const result = await projectsApi.listProjects();
      return result.data;
    },
  });
```
3. Replace `useProjects` similarly:
```typescript
export const useProjects = () =>
  useQuery({
    queryKey: queryKeys.projects(),
    queryFn: async () => {
      const result = await projectsApi.listProjects();
      return result.data;
    },
  });
```
4. Update `useProject` to use cloud API:
```typescript
export const useProject = (id: string) =>
  useQuery({
    queryKey: queryKeys.project(id),
    queryFn: () => projectsApi.getProject(id),
    enabled: !!id,
  });
```
